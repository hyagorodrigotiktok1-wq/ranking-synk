📊 Como atualizar o ranking

O ranking precisa ser atualizado pelo seu navegador (Chrome), pois os dados vêm do Instagram onde você está logado. Siga os passos abaixo — leva menos de 2 minutos.

1

Arraste o botão abaixo para a barra de favoritos

Faça isso uma vez só. Se a barra de favoritos não estiver visível, pressione Ctrl+Shift+B.

Arraste este botão para sua barra de favoritos:

📊 Synk Ranking
2

Abra o Instagram no Chrome

Acesse instagram.com e certifique-se de estar logado em qualquer conta.

3

Clique no favorito "📊 Synk Ranking"

Um painel vai aparecer na tela. Insira o GitHub Token e clique em "Coletar e publicar". Pronto — o ranking é atualizado em segundos.

🏆 Ver ranking 🔑 Criar GitHub Token
GitHub Token: acesse o link acima, marque o escopo repo, gere o token e cole no painel quando aparecer. O token fica salvo no navegador — você só precisa inserir uma vez.
